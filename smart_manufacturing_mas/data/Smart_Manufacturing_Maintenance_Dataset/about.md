# Smart Manufacturing Maintenance Dataset (SMMD)

Place your `smart_maintenance_dataset.csv` file in this directory.

**Expected columns (example):**
- `Machine_ID` — identifier
- `Temperature`, `Vibration`, `Pressure`, `Acoustic_Level` — sensor readings
- `Inspection_Duration`, `Technician_Availability`, `Downtime_Cost` — operational
- `Failure_Probability` — continuous target (regression)
- `Maintenance_Priority` — categorical target: High / Medium / Low (classification)

**Source:** Kaggle — Smart Manufacturing Maintenance Dataset
