# 6G-Enabled Intelligent Manufacturing Resource Dataset (6GMR)

Place your dataset CSV file in this directory.

**Expected columns (example):**
- `Machine_ID` — identifier
- `Temperature`, `Vibration`, `Power_Consumption` — equipment parameters
- `Network_Latency`, `Packet_Loss_Rate`, `Reliability_Score` — network indicators
- `Production_Speed`, `Quality_Metric` — process variables
- `Efficiency_Status` — target: High / Medium / Low (classification or regression)

**Source:** Kaggle — 6G-Enabled Intelligent Manufacturing Resource Dataset

**Supported tasks:** Classification, Regression, Anomaly Detection
